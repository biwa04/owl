## Scrapboxに記録するBot

Discordに投稿される文字列をScrapboxに記録するDiscordBotです。

## 使用方法

### 文字の記録を開始する

```discord-command
/start-reading
```

### 記録された文字をScrapboxに保存する

```discord-command
/save {sid} {project-id}
```
